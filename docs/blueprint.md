# **App Name**: AffairsConnect

## Core Features:

- Public Announcement Page: Dedicated section to publish announcements for all students.
- Student Record Management: System to manage and update student records, including tracking volunteer hours.
- Complaint Management: Dashboard to review and approve complaints; automated student notification system; automatic fining system upon repeated offenses; portal access control.
- Counseling Services: Feature to schedule and manage student counseling sessions.
- Fined Student Analytics: Dashboard providing visual representations of fined students, filterable by department.
- Complaint Registration and Tracking: Form for teachers to register complaints with predefined violations and status tracking.
- Automated Notifications: The system automatically generates notifications via email, SMS, and WhatsApp bot. An LLM tool will determine, according to institutional policy, whether each student notification requires corresponding parental notification, in order to improve family engagement in appropriate cases.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and reliability.
- Background color: Light blue (#E3F2FD) for a calm, unobtrusive background.
- Accent color: Purple (#7E57C2) for subtle call-to-action emphasis.
- Body and headline font: 'PT Sans', a sans-serif providing modern feel with warmth. 
- Use simple, consistent icons throughout the application for intuitive navigation.
- A clean, tab-based layout should ensure effortless navigation for both admins, teachers and students. Use of white space will separate elements to provide scannability of text.
- Subtle animations on button hovers, menu transitions and data loading to offer positive feedback. Avoid distracting, moving content.
